Deploy-guide (GitHub Pages / Netlify)
1) Last ned alle filene i denne mappen.
2) Legg dem i et repo (GitHub) eller dropp mappen rett i Netlify.
3) GitHub Pages: Settings → Pages → Source: main / root.
4) Åpne https://ditt-domene/ (index.html lastes).
5) Safari (iPhone): Del → Legg til på Hjem-skjermen.
Alt er offline-klar via service worker (sw.js).
